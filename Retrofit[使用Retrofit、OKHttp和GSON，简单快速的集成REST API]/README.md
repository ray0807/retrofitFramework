AndroidRapidGsonParsing
=======================

Tech demo for how to use GSON to rapidly parse JSON data for Android

注意：
1、将assets中的图片复制到SD卡根目录中,可以调通测试接口；实际具体项目中请删除；

      测试接口用到的工具类，一般项目中可以不用，可以直接删除；
  
3、实际具体项目中请将此3条中文文字删除。
